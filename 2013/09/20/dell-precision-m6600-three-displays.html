Dell Precision M6600 working with three monitors

I set up a triple-monitor configuration on a Dell Precision M6600 laptop workstation equipped with an ATI Fire Pro M8900 graphics chipset and docking station.

Challenge Encountered

Initial attempts to connect two external displays using DVI cables failed. When you use both DVI ports it blocks the display of the notebook itself. This prevented the desired three-display setup (laptop plus two 22" monitors).

Solution

The breakthrough came through using DisplayPort connections. I found that a genuine DisplayPort connection on the monitor worked where DVI did not.

Notably, attempts to use a DisplayPort-to-DVI adapter produced identical failures to standard DVI connections.

Result
